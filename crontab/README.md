javascript (typescript) crontab
====

元々は同僚に頼まれて作成したもの。
javascriptで定期的にタスクを実行したり、
csvを読み込んでチカチカさせたりするもの。

## Usage
### setup
```
$ npm init
$ grunt watch
```

### access
http://[Your IP]/index.html
